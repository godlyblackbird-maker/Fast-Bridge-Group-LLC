Edit the Twilio no-replies follow-up script and preview it with the active property before copying.

Body placeholders: {propertyAddress}, {status}, {agentFirstName}, {agentName}, {senderName}, {timeGreeting}, {dayGreeting}. Aliases also supported here: {firstName}, {fullName}, and {phoneNumber}. Add {timeGreeting} anywhere to place the good-time-of-day line where you want it. Add {dayGreeting} anywhere to place Happy Monday!/Tuesday/etc. based on the current day. If you leave {timeGreeting} out, no time-of-day greeting is added.
