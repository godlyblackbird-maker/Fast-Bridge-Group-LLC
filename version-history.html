Latest Release

Version 1.3.2

This release sharpens admin workflow visibility, folds planner work into the calendar experience, improves MLS-related UI consistency, and keeps manual MLS imports from disappearing when user identity buckets change.

March 27, 2026 Workflow polish Persistence fixes Version 1.3.2

Top changes

  • Promoted the version history page to 1.3.2 and moved version 1.3.1 into the previous release stack directly underneath it.
  • Moved the My Planner workflow into the calendar date-detail view so clicking a date opens the planner instead of duplicating that tool in a separate widget.
  • Reworked Admin Controls so the sticky quick-access section sits ahead of notifications in a cleaner pinned toolbar.
  • Redesigned the MLS Imports admin card into a wider featured intake panel with better hierarchy, stronger CTA treatment, and clearer workflow copy.
  • Changed the MLS Hot Deals nav symbol from lightning to fire across every tab and page that renders that entry.
  • Updated Analytics My Profits filters to use First, 2nd, 3rd, and 4th Quarter ranges instead of week and rolling month windows.
  • Added migration coverage so manually imported MLS properties stay with the correct logged-in user instead of getting stranded in a default storage bucket.
