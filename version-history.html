Latest Release

Version 1.2.7

This release sharpens the property details experience with direct Redfin and Zillow link importing, cleaner image editing controls, cleaner image presentation, and homepage theme behavior that now matches the logged-in main menu.

March 24, 2026 Property detail workflow Image editing polish Homepage theme sync

Top changes

  • Bumped the website build version to 1.2.7 across package metadata, fallback labels, and visible login and admin version badges.
  • Added direct Redfin and Zillow link importing on Property Details so public listing information can be pulled into the active property record.
  • Moved the image add control into edit mode so the plus icon only appears beside the trash action while editing property images.
  • Removed the gray background treatment behind property images so preview and gallery views show only the image itself.
  • Aligned the homepage theme preload with the dashboard default so logged-in users now see the same main-menu visual theme on the home page.
  • Updated the version history page so the newest release reflects the current property detail and theme consistency improvements.
